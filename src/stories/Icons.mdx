The `Receipt` icon is used for noting whether an event requires prior booking to attend, while the `Wallet` icon is used for the associated cost for the event.

`Link` should only be used in the context of a list of other icons, for example the Event Card component. Otherwise, it is a redundant indicator that a link is clickable.

Where events have a specified gender which isn't specifically `Male` or `Female`, the `People` icon is used instead.

## Usage

Import these icons from the `@/components/Icons` directory.

```jsx
import { ArrowIcon } from '@/components/Icons/ArrowIcon';

function MyComponent() {
    return (
        <div>
            <ArrowIcon />
            {/* Other component content */}
        </div>
    );
}
```

Each icon can be customized with props including `className` and `fill` for setting the styles of the icon.

```jsx
<ArrowIcon className={styles.icon} fill="blue" />
```
