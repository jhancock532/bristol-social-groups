# Bristol Social Storybook

This design system documents the resources and components used to create the **[bristol.social](https://bristol.social/)** website.

Learn more about the color scheme, typography setup and iconography in the style guide section, or navigate through the components folder to see the individual components in action.

The source code for Bristol Social and this Storybook instance is available [on GitHub](https://github.com/jhancock532/bristol-social-groups).
